# YouTube Downloader MVP 작업 계획

## 프로젝트 범위

이 MVP는 사용자가 YouTube 링크를 입력하면 `yt-dlp`로 영상을 1회성으로 다운로드하고, 실행 중인 서버 메모리에서만 작업 상태를 관리한 뒤, 결과 파일을 로컬 다운로드 디렉터리에 보관하는 개인용/내부용 도구를 만드는 것을 목표로 한다.

첫 번째 버전에서는 Redis/RQ나 Celery 같은 별도 큐 시스템을 도입하지 않는다. 우선 애플리케이션 프로세스 안에서 백그라운드 작업으로 시작하고, 추후 동시성이나 자동화 요구가 커질 때 전용 worker 구조로 확장한다.

## 현재 실행 환경 메모

| 항목 | 상태 |
| --- | --- |
| 기본 `python3` | Python 3.9.6 |
| Python 3.10 이상 | Python 3.12.13 설치됨 (`python3.12`) |
| ffmpeg | 설치됨 (`/opt/homebrew/bin/ffmpeg`) |
| yt-dlp | `.venv`에 Python 패키지와 CLI 설치됨 (`yt-dlp 2026.03.17`) |

## MVP 이후로 미루는 항목
